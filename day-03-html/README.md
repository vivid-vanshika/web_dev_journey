# Day 3 – Advanced HTML Tags 🚀

## 📚 What I Learned
- Semantic layout tags: `<header>`, `<nav>`, `<section>`, `<article>`, `<footer>`
- Text formatting: `<strong>`, `<em>`, `<span>`, `<code>`
- Collapsible content with `<details>` and `<summary>`
- Multimedia: `<audio>` and `<video>` with controls
- Embedding external content with `<iframe>`
- Progress indicators: `<progress>` and `<meter>`

## 🛠️ What I Built
A single webpage that demonstrates a wide range of advanced HTML elements including semantic structure, multimedia, embedded content, and progress indicators.

## 💡 Key Takeaway
Semantic HTML makes your page meaningful — tags like `<header>`, `<article>` and `<footer>` tell the browser (and search engines) what each part of your page actually is, not just how it looks.
